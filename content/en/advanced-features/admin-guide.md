# Process Consulting AI and Management Guide
ProcessGPT's Process Consulting AI helps users design and manage immediately executable business workflows (BPMN) through everyday language (natural language) or existing internal documents (PDF, etc.) without having to learn complex tools.

## 1. Document-based Automatic Process Generation (PDF to BPMN) and Collaboration
There is no need to manually design processes and create forms (Forms) while reviewing scanned images, complex tables, or PDF documents containing internal regulations. ProcessGPT independently identifies the hidden meaning and logic in uploaded unstructured data and instantly converts it into executable automated business workflows.

### 1.2 Completion of Executability: Not a Diagram, but an 'Executable File'
It doesn't stop at simply drawing a diagram for visual inspection. The converted process is set up with key elements so that it can be immediately deployed to production without additional work from administrators.
- **Automatic Form Generation:** For each stage (activity) of the process, data forms that practitioners need to fill in are automatically generated by inferring from the document context. This dramatically reduces time-consuming UI design work.
- **Automatic Reference Data Mapping:** Data Links are automatically connected so that outputs or approval records from previous work stages are seamlessly carried over as foundational data for the next stage, perfectly preventing workflow disconnection.


### 1.3 Smart Invitation: Automatic AI Expert Assignment Based on Conversation Context
After generating the process diagram, when conducting actual work or when questions arise, there is no need to worry about which person or bot to call. The system independently determines and brings in a reliable collaboration partner.
- **Proactive Collaboration Suggestion (Agent Routing):** When you enter a question or instruction in the chat window, the 'Agent Router' running in the background analyzes the intent of the conversation and identifies the most suitable specialized agent (e.g., Work Support Agent) for the task and suggests an invitation.
- **Collaboration in a Single Communication Space:** Once the user approves the invitation, the agent joins immediately without needing to leave the tool. People and AI organically communicate within a single chat workspace, processing tasks with continuity.

## 2. Basic Usage of Process Consulting AI
This is a step-by-step guide on how to use the automation features described above in the actual admin screen.

1. **Process Consulting AI Screen**
   - Process Consulting AI uses the **top AI chat window** in the Process Definition Map.

   ![](../../../uengine-image/process-gpt/admin-guide/4.png)

   - Access the Process Consulting AI chat screen.

   ![](../../../uengine-image/process-gpt/admin-guide/5.png)

<br>

2. **Process Consulting AI Creation**  
   - Enter the desired business process content through the chat window.
   - If there is content to improve or add, enter more detailed content.
   - If there is no more content to reflect, the diagram is generated based on the entered chat content according to the business process definition.

   ![](../../../uengine-image/process-gpt/process_consulting_ai_chat2.png)

   ![](../../../uengine-image/process-gpt/process_consulting_ai_chat3.png)

<br>

3. **Process Consulting AI Save**  
   - If you want to save the created diagram, click the unlocked lock button at the top of the right chat screen to save it.
   
      ![](../../../uengine-image/process-gpt/process_consulting_ai_chat4.png)

   ![](../../../uengine-image/process-gpt/process_consulting_ai_chat5.png)

   - The saved business process definition can be checked on the Process Definition Map.
   ![](../../../uengine-image/process-gpt/process_definition_map1.png)

   ![](../../../uengine-image/process-gpt/process_definition_map2.png)

<br>

## 4. Organization Chart Management (Organization)

1. **Organization Chart Definition Screen**
   - The organization chart definition is managed through the left chat screen and the right organization chart.

   ![](../../../uengine-image/process-gpt/organization_chart_chat1.png)

<br>

2. **Organization Chart Creation**  
   - Enter the team information and employee information you want to register through chat.
      - If the employee to be added is a new hire, enter the new hire information.
      - The new hire will be automatically registered through the received email address.
   - The created organization chart can be checked with team information and employee information.
   ![](../../../uengine-image/process-gpt/organization_chart_chat2.png)

<br>

3. **Organization Chart Modification**  
   - Users can directly modify and manage the organization chart.
   - If you want to add a team, click the + button next to the topmost company name element.
   ![](../../../uengine-image/process-gpt/organization_chart_chat3.png)

   - If you want to delete a team, click the delete button of the team you want to delete.
   - If you want to change the team name or modify team members, click the modify button of the team you want to modify.
   ![](../../../uengine-image/process-gpt/organization_chart_chat4.png)
