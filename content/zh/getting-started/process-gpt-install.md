# Process-GPT 安装指南

本指南将帮助您快速安装和配置 Process-GPT。

## 系统要求

在开始安装之前，请确保您的系统满足以下要求：

- **操作系统**: Windows 10+, macOS 10.15+, 或 Linux
- **内存**: 最少 4GB RAM (推荐 8GB+)
- **存储**: 至少 2GB 可用空间
- **网络**: 稳定的互联网连接

## 安装步骤

### 1. 下载安装包

从官方网站下载最新版本的 Process-GPT：

```bash
# 使用 npm 安装
npm install -g process-gpt

# 或使用 yarn 安装  
yarn global add process-gpt
```

### 2. 验证安装

安装完成后，验证安装是否成功：

```bash
process-gpt --version
```

### 3. 初始化配置

运行初始化命令设置您的工作环境：

```bash
process-gpt init
```
